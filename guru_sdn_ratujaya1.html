Tentang e-Lamaso

e-Lamaso merupakan platform pembelajaran dalam jaringan (daring/online) yang dapat digunakan untuk mempermudah guru dan siswa dalam kegiatan pembelajaran jarak jauh (PJJ).

e-Lamaso sendiri merupakan pengembangan lamaso versi sebelumnya. pada versi e-Lamaso ini, konten materi dapat diatur/disetting dengan mudah oleh guru tanpa harus mengetahui . Pun kepada siswa, platform e-Lamaso dapat dijadikan sarana latihan, ulangan, pengumuman nilai yang dapat dilakukan secara otomatis di web sekolah/e-Lamaso versi Siswa.

e-Lamaso ada 2 versi. Versi pertama adalah versi guru (e-Lamaso Guru). Versi kedua adalah versi siswa (e-Lamaso Siswa). Masing-masing memiliki keterhubungan yang sama. e-Lamaso guru (seperti lamaso saat ini yang digunakan) oleh guru untuk mengupload materi, mengoreksi soal uraian (Poto/audio/video tugas siswa), memberikan nilai, dan memberikan akses pengumuman. Sementara untuk e-Lamaso Siswa, peserta didik dapat membaca, mengerjakan tugas, mengupload tagihan tugas baik dalam bentuk poto, audio, ataupun video. Untuk tagihan tugas berupa video kemungkinan akan ada fitur upload di youtube via e-Lamaso di chanel sekolah, tetapi fitur ini masih dalam pengembangan. Melalui e-Lamaso Siswa juga, peserta didik dapat mengetahui pengumuman (nilai/pengumuman lainnya) yang telah diberikang guru pada e-Lamaso Guru.

Platform e-Lamaso

Sebagai platform, e-Lamaso dapat diterapkan di web sekolah maupun sebagai file html. Versi yang Anda pakai saat ini adalah platform file html. Sedangkan untuk platform web dapat diakses di web sekolah masing-masing setelah sekolah/admin e-Lamaso memberikan alamat link-nya. Versi Web dan file memungkinkan tampilannya berbeda namun akses penggunaannya akan tetap sama.

Server Data

e-Lamaso menjadikan Google Drive sebagai server sebagai tempat penyimpanan file. SpreadSheet (Excel Google) sebagai perekaman data. File dan data akan otomatis diakses oleh e-Lamaso.

Data siswa bersumber dari file Dapodik sekolah sehingga dapat memudahkan Operator Sekolah mengetahui kevalidan data. Misalnya, nama siswa yang salah, tempat tanggal lahir yang kurang tepat, dan lain-lain.

Membuat file Unggahan e-Lamaso

Sebelum Anda mengungah pembelajaran, Ada baiknya Anda perhatikan berikut di dalam file Notepad Anda:
Petunjuk pembuatan materi:
  1. Buatlah materi di Ms. Word seperti biasa.
  2. Salin semua teks di NOTEPAD
  3. Beri kode pada awal kalimat untuk dapat dibaca e-Lamaso.

    Berikut kode utama Lamaso yang ditulis di file Notepad:
    1. Kode Judul:
      _JUDUL_ Ditulis judulnya ⇣
      ⇣ = enter/ganti baris
      Contoh: _JUDUL_ RANGKUMAN SOAL IPA TEMA 1 KELAS 3

    2. Kode Link Gambar:
      _GAMBAR_link gambarnya
      Contoh: _GAMBAR_https://1.bp.blogspot.com/-wWPZEohYj_k/Xe_ZUEPgB3I/AAAAAAAAOgY/7idyYLxlmqgwHo6TJ9Q
      QK0sC_MQgmBXrwCLcBGAsYHQ/s1600/logo%2Blamaso.jpg

      Hasil tampilannya:

      Catatan: link di atas dapat Anda cari di Google dengan awalan https://. Usahakan link dengan awalan https:// dan hindari link dengan awalan: data:image/jpeg;base64,.

    3. Kode Link Gambar dari Google Drive:
      _GAMBAR-DRIVE_link gambar yang disimpan di Google Drive.
      Contoh: _GAMBAR-DRIVE_https://drive.google.com/file/d/1J0NUwTrxFBZ0JZBCzVTlsDFeXbn3mIci/view
      Hasil tampilannya:

      Catatan: Untuk dapat dilihat sebagai gambar dari Google Drive, usahakan pengaturan share file gambarnya diatur publik.

    4. Kode Youtube:
      _YOUTUBE_Link Youtube
      Contoh: _YOUTUBE_https://www.youtube.com/watch?v=q_7Y24A6oFY
      Hasil Tampilannya:

    5. Kode Tabel:
      tandai awal sebagai tabel:
      _START-TABEL_ ⇣
      tanda garis batas pada header
      <|HEADER|>
      tanda garis batas bukan header
      <|>
      akhiri tabel dengan tanda:
      _SELESAI-TABEL_ ⇣
      Contoh penulisan di Notepad: 
      _START-TABEL_ ⇣
    NO <|HEADER|>	NAMA<|HEADER|>KELAS ⇣
    1    <|>      ADZKA      <|> 3A ⇣
    2    <|>      NUHAA      <|> 4A ⇣
                                                        
    _SELESAI-TABEL_ ⇣
      ⇣ = enter/ganti baris
      Hasil tampilannya:
      NO NAMA KELAS
      1 ADZKA 3A
      2 NUHAA 4A

    6. Kode Soal Pilihan Ganda:
      • Pertanyaan:_PG_(No Soal)<SPASI>Teks pertanyaan ⇣
      • Opsi jawaban A: _OPSI-PG_(No Soal)A<SPASI>Teks opsi A pertanyaan ⇣
      • Opsi jawaban B: _OPSI-PG_(No Soal)B<SPASI>Teks opsi B pertanyaan ⇣
      • Opsi jawaban C: _OPSI-PG_(No Soal)C<SPASI>Teks opsi C pertanyaan ⇣
      • Opsi jawaban D: _OPSI-PG_(No Soal)D<SPASI>Teks opsi A pertanyaan ⇣
      ⇣ = enter/ganti baris
      Contoh penulisan Soal Nomor 1 di Notepad:
      _PG_1 Teks pertanyaan
      _OPSI-PG_1A Teks opsi A pertanyaan
      _OPSI-PG_1B Teks opsi B pertanyaan
      _OPSI-PG_1C _GAMBAR-DRIVE_https://drive.google.com/file/d/1J0NUwTrxFBZ0JZBCzVTlsDFeXbn3mIci/view
      (contoh opsi berupa gambar dari google drive)
      _OPSI-PG_1D Teks opsi D pertanyaan
      Hasil tampilannya (Coba klik salah satu opsi untuk melihat animasinya):
      Ini adalah pertanyaan soal pilihan ganda.

    7. Kode Soal Pilihan Ganda Dalam bentuk Tabel
      Adakalanya, soal pilihan ganda menampilkan opsi jawaban yang dikemas dalam bentuk tabel.
      Sintak kode:
      _PG_(No soal)<SPASI>Yang termasuk contoh simbiosis dan contohnya yang tepat adalah ... ⇣
      _START-TABEL-OPSI_ ⇣
      "spasi" <|HEADER|> Simbiosis <|HEADER|> Contoh ⇣
      _OPSI-SEL_(No soal)A <|> Mutualisme <|> Hubungan bunga dengan lebah ⇣
      _OPSI-SEL_(No soal)B <|> Parasitisme <|> Hubungan bakteri E-Coli pada usus manusia ⇣
      _OPSI-SEL_(No soal)C <|> Komensialisme <|> Benalu pada batang pohon mangga ⇣
      _OPSI-SEL_(No soal)D <|> Xerofitisme <|> Hubungan burung jalak dengan kerbau ⇣
      _SELESAI-TABEL-OPSI_ ⇣
      ⇣ = enter/ganti baris
      Berikut ini contoh penulisan di Notepad, contoh di sini No soal adalah 5:
          _PG_5<SPASI>Yang termasuk contoh simbiosis dan contohnya yang tepat adalah ... 
    _START-TABEL-OPSI_
    "spasi" <|HEADER|>	Simbiosis	<|HEADER|>	Contoh
    _OPSI-SEL_5A	<|>	Mutualisme	<|>	Hubungan bunga dengan lebah
    _OPSI-SEL_5B 	<|>	Parasitisme	<|>	Hubungan bakteri E-Coli pada usus manusia
    _OPSI-SEL_5C 	<|>	Komensialisme	<|>	Benalu pada batang pohon mangga
    _OPSI-SEL_5D 	<|>	Xerofitisme	<|>	Hubungan burung jalak dengan kerbau
    _SELESAI-TABEL-OPSI_

      Dan ini hasilnya:
      1. Yang termasuk contoh simbiosis dan contohnya yang tepat adalah ...
        Simbiosis contoh
        Mutualisme Hubungan bunga dengan lebah
        Parasitisme Hubungan bakteri E-Coli pada usus manusia
        Komensialisme Benalu pada batang pohon mangga
        Xerofitisme Hubungan burung jalak dengan kerbau

    8. Kode Soal Uraian/Essay (Jawaban berupa Poto):
      _ESSAY-NO_(No Soal)<SPASI>teks pertanyaan soal uraian/Essay.
      Contoh penulisan 5 soal essay dengan satu lembar poto jawaban siswa di Notepad:
      _ESSAY-NO_1 teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 1.
      _ESSAY-NO_2 teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 2.
      _ESSAY-NO_3 teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 3.
      _ESSAY-NO_4 teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 4.
      _ESSAY-NO_5 teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 5.
      Hasil tampilannya :
      1. teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 1.
      2. teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 2.
      3. teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 3.
      4. teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 4.
      5. teks pertanyaan soal uraian/Essay. No soal di isi dengan nomor soal 5.

    9. Kode Upload Tugas:
      _POTO-JAWABAN-TUGAS_1/2/3/4/5 <diakhir nomor tanpa tanda garis miring >
      Hasilnya:
      Upload Media(Poto, audio, video, pdf, word/txt, dll) jawaban Tugas No. 1, 2, 3, 4, 5

    10. Kode Simbol Matematika:
      • Pecahan Biasa, contoh = :
        _PECAHAN-BIASA_a/b

      • Pecahan Campuran, contoh = :
        _PECAHAN-CAMPURAN_a/b/c

      • Akar Pangkat dua(Akar Kuadrat), contoh = :
        _AKAR-KUADRAT_angka

      • Akar Pangkat tiga, contoh = :
        _AKAR-TIGA_angka

      • Kode Pi (lingkaran), contoh = :
        _PHI_

      • Kode pangkat, contoh 8 pangkat 3 = :
        _PANGKAT_a/b

      • Kode Matematika lainnya yang dapat dibuat di equation editor online, contoh = :
        _EQUATION-LAINNYA_\lim_{x \to a} \frac{f(x) - f(a)}{x - a} (kode diambil dari link di atas tanpa spasi)

      Contoh penulisan Kode Matematika di Notepad:
      _PECAHAN-BIASA_1/2
      _PECAHAN-CAMPURAN_3/1/4
      _AKAR-KUADRAT_144
      _AKAR-TIGA_64
      _PHI_
      _PANGKAT_15/2
      _EQUATION-LAINNYA_\sqrt{325\frac{2}{3}}
      Hasil tampilan:
      Pecahan Biasa
      Pecahan Campuran
      Akar Kuadrat
      Akar pangkat tiga
      kode Phi
      15 pangkat 2
      kode equation lainnya: \sqrt{325 \frac{2}{3}}

    11. Kode lainnya dapat dibuat dengan format HTML: contoh:
      <div style="border:1px solid blue; background-color:yellow; color: red">

      ....... tulisan Anda ....

      </div>
      Hasilnya akan seperti ini:
      ....... tulisan Anda ....
      Ingin mengetahui lebih dalam format penulisan bahasa HTML, Anda dapat mengunjungi link www.w3schools.com

Petunjuk Menu e-Lamaso Guru

Daftar Kelas

Menu ini menampilkan data kelas guru setelah login. Pada data kelas, guru yang bersangkutan dapat mengedit, menghapus, dan/atau menambahkan peserta didiknya.

Guru juga dapat mencetak data kelas ini di perangkat (device) printer yang terhubung. Anda disarankan menggunakan aplikasi peramban (browser) dari Google Chrome untuk mendapatkan hasil yang optimal (meski peramban lain pun bisa mengaksesnya). Hasil cetak (printout) secara bawaan (default) dengan ukuran kertas A4 dan berorientasi landscape (tergantung laman yang akan dicetak).

Pembelajaran

Pada laman ini, Guru dapat membuat kontent KBM dan melihat riwayat kontennya. Terdapat 3 fitur utama, yang pertama fitur mengupload materi (rangkuman, soal, ulangan, dll) dengan segala akses preview dan tanggal akses siswanya. Fitur kedua berisi tabel mengenai riwayat pemberian materi dengan segala bentuk tanggal akses kepada siswanya. Fitur ketiga, berisi tabel keaktifan siswa dengan segala fitur interaksi dengan siswanya via WhatsApp atau yang lainnya.
